# Підтримка NUMA та Tiered Memory у .NET 10 GC

.NET 10 впроваджує підтримку **NUMA (Non-Uniform Memory Access)** та багаторівневої пам’яті (Tiered Memory). Це дозволяє:

- Розподіляти об’єкти у різні фізичні блоки пам’яті залежно від частоти доступу;
- Переміщати "холодні" об’єкти на повільні NVMe SSD або повільніший оперативний блок, звільняючи швидку RAM;
- Оптимізувати продуктивність багатоядерних систем та хмарних сервісів.

### Переваги:

- Зменшення пауз GC для активних об’єктів;
- Зниження латентності при високих навантаженнях;
- Підвищення ефективності використання оперативної пам’яті та SSD.
