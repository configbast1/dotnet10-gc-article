# Оптимізація Large Object Heap (LOH) у .NET 10

LOH використовується для об’єктів розміром понад 85 KB. У .NET 10:

- Введено **паралельне очищення LOH**;
- Динамічна компактація LOH при високому навантаженні;
- Зменшено фрагментацію, що покращує продуктивність пам’яті.

### Вплив на реальні додатки:

- Продуктивність додатків з великими масивами та об’єктами;
- Зменшення витоків пам’яті у long-running сервісах;
- Підвищення стабільності AI/ML додатків у ML.NET.
